
# RC Car

## About
This project is a simple remote-controlled car that I built using a robotics kit. It is lightweight and designed to move quickly on smooth surfaces. The car can also be modified with different wheels to drive on rougher terrain.

## Features
- Remote controlled
- Lightweight design
- Fast movement
- Can be adapted for different terrains

## Parts Used
- Robo Robo Kit
- Motors
- Wheels
- Chassis
- Remote controller
- Battery pack

## How It Works
The remote sends signals to the controller on the car, which powers the motors to move the wheels. The motors allow the car to drive forwards, backwards, and turn left or right.
